Business Case No - 2

Dynamic Patient Scheduling System

Business case Description

A Dynamic Patient Scheduling System is essential to address inefficient scheduling in hospitals caused by outdated methods. Current systems fail to prioritize patient urgency or adapt to real-time changes in doctor availability, leading to long wait times, missed care, and underutilized resources.

This system will use modern technology to dynamically schedule appointments based on patient needs and doctor schedules, ensuring urgent cases are prioritized, reducing delays, and improving resource utilization. By enhancing efficiency and access to care, it will significantly improve patient satisfaction and hospital operations.

Algorithms and Core Concepts

Patient Prioritization
  • Priority Queues (Min-Heap):
    Dynamically prioritize patients based on urgency and severity to ensure critical cases are scheduled first.
  • Triage Mechanism:
    Categorize patients into priority levels to streamline scheduling based on their condition.
Real-Time Scheduling
  • Graph Algorithms (Maximum Bipartite Matching):
    Match patients to doctors and time slots efficiently by modeling them as a bipartite graph.
  • Hash Maps:
    Quickly map doctors to their availability and retrieve schedules in real time.
Resource Optimization
  • Greedy Algorithm:
    Allocate appointments and resources efficiently by focusing on minimizing delays and conflicts.
  • Dynamic Programming:
    Optimize overlapping appointments and resource utilization by solving smaller sub-problems.

Algorithm and Workflow

Step 1:
Input Collection
  • Collect patient details (e.g., name, condition severity, and urgency level).
  • Gather doctor availability (e.g., time slots, specialization).
  • Retrieve hospital resources (e.g., consultation rooms, diagnostic equipment).
Step 2:
Patient Prioritization (Using Priority Queue)
  • Create a priority queue where
    • Patients with critical conditions or emergencies are assigned higher priority.
    • Priority = Severity Level * Weightage + Waiting Time.
  • Insert all patient requests into the priority queue.
  • Extract patients from the queue in descending order of priority.
Step 3:
Doctor-Patient Matching (Using Bipartite Graph and Maximum Matching)
  • Represent doctors and patients as nodes in a bipartite graph:
    • Patients on one side and doctors on the other.
    • Add edges between compatible pairs based on doctor specialization and patient needs.
  • Apply Maximum Bipartite Matching to allocate patients to available doctors optimally.
Step 4:
Time Slot Allocation (Using Greedy Algorithm)
  • For each matched doctor-patient pair
    • Select the earliest available time slot for the doctor.
    • Assign the patient to this time slot.
  • Ensure no overlapping appointments for the same doctor.
Step 5:
Conflict Resolution (Using Dynamic Programming)
  • Identify overlapping or conflicting appointments (e.g., emergency cases).
  • Use dynamic programming to rearrange appointments by finding the optimal combination of non-overlapping intervals.
Step 6:
Resource Allocation (Using Hash Maps)
  • Map patients and doctors to available resources like consultation rooms and equipment using a hash map.
  • Ensure efficient allocation by checking resource availability in real time.
Step 7:
Real-Time Adjustments
  • If a doctor becomes unavailable (e.g., emergency or leave)
    • Re-run the bipartite matching algorithm for affected patients.
  • If a new patient request arrives
    • Insert it into the priority queue and repeat the prioritization process.
Step 8:
Output Generation
  • Display a dynamic schedule to:
    • Patients (appointment time and doctor).
    • Doctors (patient list and time slots).
    • Hospital staff (resource allocation).
  • Send automated notifications for scheduled appointments.

                #include <iostream>
                #include <queue>
                #include <vector>
                #include <unordered_map>
                #include <algorithm>
                
                using namespace std;
                
                // Structure for Patient
                struct Patient {
                    int id;
                    string name;
                    int severity;  // Higher severity = Higher priority
                    int waitingTime;
                };
                
                // Structure for Doctor
                struct Doctor {
                    int id;
                    string name;
                    int availableSlots;
                };
                
                // Priority Queue Comparator for Patient
                struct ComparePatient {
                    bool operator()(const Patient &p1, const Patient &p2) {
                        return (p1.severity + p1.waitingTime) < (p2.severity + p2.waitingTime);
                    }
                };
                
                // Function to match patients with doctors
                void matchPatientsToDoctors(priority_queue<Patient, vector<Patient>, ComparePatient> &patientQueue,
                                            vector<Doctor> &doctors) {
                    unordered_map<int, vector<int>> doctorPatientMap;  // Doctor ID -> Patient IDs
                
                    for (auto &doctor : doctors) {
                        while (!patientQueue.empty() && doctor.availableSlots > 0) {
                            Patient topPatient = patientQueue.top();
                            patientQueue.pop();
                            doctorPatientMap[doctor.id].push_back(topPatient.id);
                            doctor.availableSlots--;
                        }
                    }
                
                    // Display the scheduling results
                    cout << "\nScheduling Results:\n";
                    for (const auto &entry : doctorPatientMap) {
                        cout << "Doctor ID " << entry.first << " is assigned to Patients: ";
                        for (const auto &patientId : entry.second) {
                            cout << patientId << " ";
                        }
                        cout << endl;
                    }
                }
                
                int main() {
                    // Input: Patients
                    priority_queue<Patient, vector<Patient>, ComparePatient> patientQueue;
                    patientQueue.push({1, "John", 8, 2});
                    patientQueue.push({2, "Alice", 5, 3});
                    patientQueue.push({3, "Bob", 9, 1});
                    patientQueue.push({4, "Eve", 6, 2});
                
                    // Input: Doctors
                    vector<Doctor> doctors = {
                        {1, "Dr. Smith", 2},
                        {2, "Dr. Watson", 2}
                    };
                
                    // Perform Doctor-Patient Matching
                    matchPatientsToDoctors(patientQueue, doctors);
                
                    return 0;
                }
